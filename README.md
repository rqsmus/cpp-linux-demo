# Mini Command Parser
A small C++ demo project that reads user commands and performs actions accordingly.  
Developed and tested on Linux (WSL).  
This project demonstrates basic C++ programming skills and use of Git for version control.

## Features
- Command parsing (e.g., "on <name>", "status <name>", "help")
- Case-insensitive input handling
- Example of continuous command loop

## How to Run
1. Compile using `g++ main.cpp -o mini_parser`
2. Run with `./mini_parser`
